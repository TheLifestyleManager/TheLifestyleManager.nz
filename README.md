# TheLifestyleManager.nz

Static site for The Lifestyle Manager.

## Project structure

* `index.html`, `services.html`, `styles.css`, `robots.txt`, `sitemap.xml`: Main site assets.
* `CNAME`: Custom domain mapping for GitHub Pages.
* `assets/images/`: Place to store headshots, OG images, and any other site imagery.

## Development

Open `index.html` locally in a browser to preview changes. Deploys are handled via GitHub Pages.
